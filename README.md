## Установка и запуск

### Клонируй репозиторий

### 1. Запуск сервера

```bash
cd avito-seller-cabinet/server
npm run start
```

Сервер запустится на http://localhost:8080

### 2. Запуск фронтенда

```bash
cd avito-seller-cabinet/frontend
npm run dev
```

Приложение откроется на http://localhost:5173

### 3.Готово

Ai функции работают через внешний Api Groq
